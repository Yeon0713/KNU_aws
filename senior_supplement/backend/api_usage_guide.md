# 🏥 AI 건강 분석 서비스 사용 가이드

## ⚠️ API 사용 제한 안내

### 현재 상황
- AWS Claude API에 **분당 호출 제한**이 있습니다
- 연속으로 많은 요청을 보내면 일시적으로 차단됩니다

### 권장 사용 방법
1. **건강검진 분석**: 하루에 2-3회 정도만 사용
2. **식단 분석**: 식사 후 1회씩 (하루 3회 정도)
3. **영양제 추천**: 주 1-2회 정도만 사용
4. **팩트체킹**: 필요할 때만 사용

### 오류 발생 시 대처법
- "AI 서버 사용량이 많습니다" 메시지가 나오면
- **5-10분 기다린 후** 다시 시도하세요
- 급하지 않다면 **30분-1시간 후** 시도하는 것이 좋습니다

### 개선 방안
1. **AWS 계정 업그레이드**: 더 높은 API 한도 확보
2. **캐싱 시스템**: 동일한 요청은 저장된 결과 사용
3. **대기열 시스템**: 요청을 순서대로 처리

## 💡 팁
- 앱을 실제로 사용할 때는 문제없이 작동합니다
- 테스트를 많이 하면서 제한에 걸린 것입니다
- 실제 사용자는 이런 빈도로 요청하지 않습니다
